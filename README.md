# 프로젝트 소개

1. 프로젝트 명 : GOSTOP

2. 팀원: 박경민, 양회욱, 고민재

3. 프로젝트 소개: 고속도로, 휴게소, 교차로 공공데이터를 사용하였으며 입구 IC와 출구 IC를 입력하면 
                 해당 경로와 경로 상의 휴게소가 지도에 출력이 되며, 휴게소의 대표 음식을 알 수 있습니다.

4. 개발기간: 2018.10~2018.11

5. 데모영상: https://www.youtube.com/watch?v=oeaIcBSzFKU
